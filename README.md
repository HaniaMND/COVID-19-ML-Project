Link to dataset : https://www.kaggle.com/datasets/einsteindata4u/covid19

# Content :
1. Exploratory Data Analysis :

   - Deep Analysis
   - Initial / Target visualization
   - Variable signification
   - More Detailed Analysis

2. Data preprocessing
3. Modeling
